# Glossary

| Term | Definition/Description |
|------|------------------------|
| Gym | Organization where sports activities and training sessions take place. |
| User | Generic person interacting with the system. Can be an Instructor or Athlete. |
| Instructor | User responsible for teaching one or more modalities and managing classes. |
| Athlete | User enrolled in the gym who can join modalities, classes and make payments. |
| Modality | Type of sport or training activity practiced in the gym (e.g. Boxing, Yoga, Judo). |
| ClassSession | Scheduled training session of one modality taught by an instructor. |
| Room | Physical space where classes take place, with limited capacity. |
| Schedule | Date and time assigned to a class session. |
| Gym Opening Hours | Time interval in which the gym can operate and classes may occur. |
| Enrollment | Association of an athlete to a modality. |
| Booking | Association of an athlete to a class session. |
| Subscription Plan | Membership plan assigned to an athlete with specific benefits or limits. |
| Basic Plan | Standard subscription plan with reduced access conditions. |
| Premium Plan | Advanced subscription plan with extended access conditions. |
| Payment | Financial transaction made by an athlete to keep membership active. |
| Technical Level | Skill level of an athlete used to access advanced classes. |
| Capacity | Maximum number of athletes allowed in a room or class. |
